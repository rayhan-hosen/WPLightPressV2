## ğŸ”§ Installation

1.  **Download**: Download the `WpLightPressV2.zip` file.
2.  **Upload**: Go to your WordPress Dashboard > Appearance > Themes > Add New > Upload Theme.
3.  **Activate**: Install and activate the theme.
4.  **Setup**:
    - Go to **Appearance > Widgets** to configure your sidebar and homepage widgets.
    - Use the **Featured Post** checkbox in the post editor to add posts to the Hero Slider.
